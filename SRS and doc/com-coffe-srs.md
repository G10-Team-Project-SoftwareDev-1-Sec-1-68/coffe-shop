# เอกสารข้อกำหนดความต้องการซอฟต์แวร์ (SRS)
## ระบบจัดการร้านกาแฟพร้อมระบบออเดอร์ออนไลน์
### เวอร์ชัน 1.0 (Final Complete)

## 1. บทนำ
### 1.1 วัตถุประสงค์
เอกสารนี้อธิบายข้อกำหนดความต้องการซอฟต์แวร์สำหรับระบบจัดการร้านกาแฟที่รองรับการสั่งซื้อทั้งหน้าร้านและออนไลน์ โดยมีวัตถุประสงค์เพื่อจัดการการขาย บริหารสต็อก และสมาชิก ให้เป็นระบบเดียวกัน รองรับการทำงานต่อเนื่องแม้ไม่มีอินเทอร์เน็ต

### 1.2 ขอบเขต
ระบบประกอบด้วยส่วนการทำงานหลัก: ระบบหน้าร้าน (POS), เว็บแอปพลิเคชันสำหรับลูกค้า, ระบบสมาชิก, ระบบสต็อก, ระบบพนักงาน และระบบรายงาน

## 2. ความต้องการด้านฟังก์ชันการทำงาน

## 3. ความต้องการที่ไม่ใช่ฟังก์ชันการทำงาน
### 3.1 ประสิทธิภาพ
| ID | Requirement Description | Metric/Constraint |
| :--- | :--- | :--- |
| **PRF-01** | **Offline Mode** | ระบบ POS ต้องทำงานได้แม้อินเทอร์เน็ตขัดข้อง |
| **PRF-02** | **Response Time** | เวลาตอบสนองไม่เกิน 5 วินาที |
| **PRF-03** | **Capacity** | รองรับผู้ใช้พร้อมกันอย่างน้อย 100 คน  |
| **PRF-04** | **Availability** | ทำงานได้ 24 ชั่วโมง หรือ ตามเวลาเปิดร้าน + Maintenance  |
| **PRF-05** | **Sync Speed** | ต้อง Sync ข้อมูลกลับภายใน **60 วินาที** หลังเน็ตมา |

### 3.2 ความปลอดภัย
| ID | Requirement Description | Priority |
| :--- | :--- | :--- |
| **SEC-01** | เข้ารหัสข้อมูลส่วนบุคคล และยืนยันตัวตนด้วย OTP | High |
| **SEC-02** | **สำรองข้อมูลอัตโนมัติ** และป้องกันการเข้าถึงที่ไม่ได้รับอนุญาต | High |

### 3.3 ความสามารถในการใช้งาน
| ID | Requirement Description | Priority |
| :--- | :--- | :--- |
| **USB-01** | ใช้งานง่าย รองรับภาษาไทย **และภาษาอื่นๆ** | High |
| **USB-02** | รองรับการทำงานบนทุกอุปกรณ์  | High |
| **USB-03** | **มีคู่มือและวิดีโอสอนการใช้งาน** | Medium |
| **USB-04** | รองรับการทำงานออฟไลน์บางส่วน | High |
